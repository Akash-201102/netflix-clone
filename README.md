# Netflix Clone (HTML + CSS)

This project is a simple and clean Netflix homepage clone built using only HTML and CSS. It includes a fixed navbar, a hero section with a background banner, movie rows with horizontal scrolling, and a minimal dark-themed UI. This clone focuses purely on frontend design and layout practice.

* Features
- Netflix-style dark UI and red accent theme  
- Fixed navbar with smooth layout  
- Hero banner with main headline and call-to-action button  
- Horizontally scrollable movie rows  
- Hover zoom effect on movie thumbnails  
- Clean and modern UI using only HTML + CSS  
- Fully static, no JavaScript or backend

* Tech Stack
- **HTML5**
- **CSS3 (Flexbox, background images, responsive structure)**

* How to Run
1. Download the project files  
2. Keep all images and the HTML file in the same folder  
3. Open `index.html` in any browser  
4. The site loads instantly (no setup needed)
